Freetalks
=========

Freetalks is a catalog of free talks and lectures.

### Development Data

Run tests:

    nosetests --with-gae --gae-lib-root=/PATH/TO/google_appengine

Go to [Development Console][console] and run the follow:

    from tests.fixtures import dataset
    dataset.setup()
